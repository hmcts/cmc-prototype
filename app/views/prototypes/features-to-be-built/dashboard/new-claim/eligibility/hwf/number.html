Do you have a Help with Fees reference number?

You'll only have one if you've already applied for Help with Fees in respect of this claim.

Decide whether to apply for Help with Fees

Apply for Help with Fees (opens in a new tab) and make a claim using a Help with Fees number. If you need to use the paper Help with Fees application rather than the online version then you will not be able to use Online Civil Money Claims to issue your claim.

When you apply for Help with Fees you will be asked for the number on your court or tribunal form. Please note that this as N1.

Make a note of the Help with Fees number when you make an application.

You will need this number to make your claim when you are returned to this service.
