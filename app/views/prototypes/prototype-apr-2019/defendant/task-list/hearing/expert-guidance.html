Using an expert in small claims

It's rare for judges to allow you to use an expert in a small claim.

You must ask for permission to use an expert - a judge will then decide whether the expert is necessary.

The judge will also decide whether the cost of the expert is reasonable compared to the value of the claim.

Even though you and the {% if data['party'] == 'defendant' %}claimant{% else %}defendant{% endif %} share the cost of hiring the expert, judges are still responsible for keeping costs in proportion to value.

When an expert is allowed

A judge will only allow you to use an expert if they believe they need the expert's specialist knowledge to rule on the claim.

For example, they might allow a structural engineer to be used in a claim about building works or a mechanic in a claim about car repairs.

If a judge allows you to use an expert, you'll usually have to share one expert with the {% if data['party'] == 'defendant' %}claimant{% else %}defendant{% endif %}.

The expert will provide a written report - they won't come to court in person.

Paying for the expert

The cost of hiring an expert depends on which expert you and the {% if data['party'] == 'defendant' %}claimant{% else %}defendant{% endif %} choose.

You both share this cost.

If you get permission to use an expert

First, you and {% if data['party'] == 'defendant' %}the claimant{% else %}the defendant{% endif %} need to follow these steps:

  1. Make a list of 3 independent experts who can provide a written report in return for a fee.
  2. Avoid discussing the case in detail with the 3 experts.
  3. Tell the court the names and costs of the 3 experts.

{% if data['party'] == 'defendant' %}You then pick{% else %}The defendant then picks{% endif %} one expert from the list. After that, both of you need to follow these steps:

  1. Provide questions for the expert to answer in a report.
  2. Pay the expert, sharing the cost equally.

Finally, {% if data['party'] == 'defendant' %}the claimant needs{% else %}you need{% endif %} to:

  1. Upload the expert's report.
  2. Send the expert any questions either of you have about the report.
  3. Upload the expert's response.
